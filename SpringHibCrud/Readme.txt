Тестовое задание CRUD.
Есть одна таблица books в базе данных test. Скрипт инициализирующий и
заполняющий таблицу лежит в файле TableQuery. Инициализируется прямо в
Intellij idea Ultimate(создается подключение копируется в поле запросов и выполняется). 
Само подключение приложения находится в файле BookFac.
Используемые технологии:
- Maven;
- Spring;
- MySQL jdbc connector (вместо Hibernate);
- База данных соответственно MySQL.
- В качестве вывода используются JSP(JSTL(core), EL);
Поиск выполнен по полю title, но не сложно реализовать и любой другой.
